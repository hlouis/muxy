# Code blocks

Inline: `let x = 1`.

Fenced (no language):

```
plain text
  indented line
```

Fenced (swift):

```swift
import Foundation

func greet(_ name: String) -> String {
  return "Hello, \(name)!"
}
```

Fenced (json):

```json
{
  "a": 1,
  "b": [true, false, null],
  "c": "string"
}
```

Indented code block:

    line one
    line two
